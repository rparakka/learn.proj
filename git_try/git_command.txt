for pushing the changes to central repository

to add files from local workspace to local repository  (index it)

# git add
# git add -A  - to add all files from workspace to local repo index
# git status  - to know the status of files in local repository      
# git commit -m "commit comments" - to commit the changes from local workspace to local repository

# git log - to see all the changes , date / time , author etc

# git commit -a - m   to commit all changes

from master branch we can create 1st branch
# git branch firstbranch

to switch to branch to firstbranch
# git checkout firstbranch

# git merge firstbranch

# git rebase master
way of combining work between different masters
